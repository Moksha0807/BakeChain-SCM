# BakeChain SCM

BakeChain SCM is a role-based Bakery Supply Chain Management System developed using PHP and MySQL. It manages suppliers, raw materials, products, production batches, inventory, customer orders, deliveries, tracking and reports.

## Technology Stack

- PHP
- MySQL
- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- Bootstrap Icons
- MySQLi
- Apache using XAMPP

## User Roles

- Admin
- Supplier
- Production Manager
- Delivery Partner
- Customer

## Main Modules

- Supplier Management
- Raw Material Management
- Product Management
- Production Batch Management
- Inventory Management
- Order Management
- Delivery Management
- Tracking and Reports

## Unique Features

- QR-based batch verification
- Smart alternate supplier suggestion
- Supply-chain bottleneck detection
- Order and delivery tracking
- Role-based dashboards

## Local Installation

1. Install XAMPP.
2. Copy the project folder into `C:\xampp\htdocs`.
3. Start Apache and MySQL.
4. Create a database named `cookie_scm`.
5. Import the provided SQL database file.
6. Copy `config/database.example.php` as `config/database.php`.
7. Add the correct database credentials.
8. Open:

   `http://localhost/cookie_scm/`
